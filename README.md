# Dialog Tree Editor

Визуальный редактор диалоговых деревьев для веб-проектов.

## Возможности
- Создание узлов диалога
- Связывание реплик
- Drag-and-drop редактирование
- Экспорт и импорт в формате JSON

## Технологии
- Svelte
- SVG
- JavaScript

## Назначение
Проект разработан в рамках курсовой работы и предназначен для упрощения создания нелинейных диалогов в играх и обучающих системах.
