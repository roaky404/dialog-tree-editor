# Dialog Tree Editor

Визуальный плагин для редактирования диалоговых деревьев в веб-среде, предназначенный для упрощения проектирования нелинейных сценариев в интерактивных системах.

Проект разработан в рамках курсовой работы по дисциплине  
«Инструментальные средства разработки программного обеспечения с открытым исходным кодом».

## Описание проекта

Dialog Tree Editor представляет собой веб-плагин, позволяющий визуально создавать, редактировать и анализировать диалоговые деревья. Пользователь может работать с узлами диалога в наглядной графической форме, связывать реплики, редактировать текст и управлять структурой сценария без необходимости ручного редактирования файлов.

Редактор ориентирован на использование в играх, образовательных платформах, чат-ботах и других системах, где требуется реализация ветвящихся диалогов.

## Цель разработки

Целью проекта является создание лёгкого, расширяемого и удобного инструмента для визуального редактирования диалоговых деревьев, не привязанного к конкретному игровому движку или платформе.

## Основные возможности

- визуальное создание узлов диалога;
- соединение узлов с помощью связей;
- перемещение элементов с помощью drag-and-drop;
- редактирование текста реплик;
- автоматическое обновление связей при изменении структуры;
- экспорт и импорт диалоговых деревьев в формате JSON;
- проверка базовой целостности структуры.

## Запуск проекта

Клонировать репозиторий:
git clone https://github.com/roaky404/dialog-tree-editor.git
Установить зависимости:
npm install
Запустить проект:
npm run dev
